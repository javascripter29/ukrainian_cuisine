# Ukrainian Cuisine 🇺🇦

## Веб-приложение для рецептов украинской кухни

**Ukrainian Cuisine** - это аккуратное и современное приложение для хранения, просмотра и управления рецептами украинской кухни. Проект помогает собрать блюда, ингредиенты и базовую информацию о приготовлении в одном удобном цифровом каталоге.

Приложение подойдет для кулинарного проекта, небольшого гастрономического сервиса, обучающей платформы, внутреннего каталога рецептов или MVP для дальнейшего развития продукта.

## Что в себе представляет ✨

- 🍲 **Каталог рецептов** - блюда отображаются в виде понятных карточек.
- 🧄 **База ингредиентов** - ингредиенты хранятся отдельно и могут использоваться в разных рецептах.
- ⚖️ **Количество продуктов** - для каждого рецепта можно указать нужное количество ингредиентов.
- 🖼️ **Изображения блюд** - у рецепта может быть ссылка на изображение.
- 🔐 **Авторизация пользователей** - регистрация и вход по email и паролю.
- 🛡️ **Защищенные разделы** - доступ к части функциональности ограничен авторизацией.
- 🧾 **Валидация форм** - данные проверяются перед сохранением.
- 🗄️ **Надежное хранение данных** - рецепты, ингредиенты и пользователи сохраняются в PostgreSQL.

## Почему это удобно для бизнеса 💼

Проект уже имеет основу полноценного CRUD-приложения: можно добавлять данные, хранить их в базе, связывать рецепты с ингредиентами и развивать интерфейс под реальные бизнес-задачи.

Такой формат легко масштабировать:

- добавить поиск по блюдам;
- сделать фильтры по категориям;
- подключить загрузку изображений;
- добавить личные кабинеты;
- реализовать избранные рецепты;
- посчитать себестоимость блюда;
- подготовить публичную витрину для посетителей;
- добавить административную панель для контент-менеджера.

## Технологическая основа 🚀

Проект построен на современном и востребованном стеке:

- **Next.js 16** - быстрый React-фреймворк для веб-приложений;
- **React 19** - современный интерфейс;
- **TypeScript** - надежность и предсказуемость кода;
- **Prisma** - удобная работа с базой данных;
- **PostgreSQL** - стабильное хранение данных;
- **NextAuth** - авторизация пользователей;
- **HeroUI** - готовые UI-компоненты;
- **Zustand** - управление состоянием;
- **Zod** - проверка данных;
- **Tailwind CSS** - гибкая стилизация интерфейса.

## Основные разделы 🧭

- **Главная страница** - список рецептов.
- **Добавление рецепта** - форма с названием, описанием, изображением и ингредиентами.
- **Ингредиенты** - раздел для управления продуктами.
- **О проекте** - информационная страница о тематике украинской кухни.
- **Авторизация** - вход и регистрация пользователей.

## Ценность проекта 🌟

Это не просто техническая заготовка, а хорошая база для продукта, который можно довести до коммерческого уровня. В нем уже заложены ключевые сущности: пользователи, рецепты, ингредиенты и связи между ними.

## Локальный запуск для демонстрации 🛠️

Если нужно показать проект локально, используется стандартный запуск:

```bash
npm install
docker compose up -d
npm run prisma:generate
npx prisma migrate dev
npm run dev
```

После запуска приложение доступно по адресу:

```text
http://localhost:3000
```
